kali 2.0 玩机姿势

Post at: 2015-12-22 Share Like Views

先来一个全面更新～
apt-get clean
apt-get update
apt-get upgrade
apt-get dist-upgrade
我觉得不要一味的去找阿里，中科大这些国内源，如果你觉得官方源的速度还可以，就不要换了，官方的是最可靠的。

卸载自带的iceweasl，安装firefox并汉化。
apt-get remove iceweasel
echo -e “\ndeb http://downloads.sourceforge.net/project/ubuntuzilla/mozilla/apt all main” | tee -a /etc/apt/sources.list > /dev/null

apt-key adv –recv-keys –keyserver keyserver.ubuntu.com C1289A29

apt-get update
apt-get install firefox-mozilla-build
这样就装好了，然后去这下载中文包http://ftp.mozilla.org/pub/firefox/releases/
找到自己的版本，系统，在xpi文件夹有个zh_cn.xpi文件，下载就会提示安装了。

安装Flash插件：
apt-get install flashplugin-nonfree
update-flashplugin-nonfree –install
安装shadowsocks代理工具
apt-get install python-pip
pip install shadowsocks
相关配置：shadowsocks详细配置

安装搜狗输入法～
apt-get install fcitx
到搜狗官网下载安装包：搜狗拼音for linux
cd /root/下载
dpkg -i 文件名.deb
reboot
配置神器vim～
贴出我的神级配置，研究了一天并借鉴了网上部分人的代码。
打开终端：
vim .vimrc //vim配置文件
//内容如下：
syntax on
set number
set nobackup
set cursorline
set autoindent
set cindent
set tabstop=4
set shiftwidth=4
set smartindent
set mouse=a
set showmatch
set incsearch
set nocompatible
set history=400
set makeprg=g++\ %\ -o\ /tmp/%<
map ggvG”+y
map ggvG
map ggvG=

map :call CompileRunGpp()
func! CompileRunGpp()
exec “w”
exec “!clear”
exec “!g++ % -o /tmp/%<.exe”
exec “! /tmp/%<.exe”
endfunc

map :call Compile()
func! Compile()
exec “w”
exec “make”
exec “cw”
endfunc
实现了：
Ctrl+C 复制当前所有内容
Ctrl+A 全选
Ctr+Z 格式化所有代码（不是删除的意思。。）
F６ 编译，如有错误调用quickfix 可以快速查看错误的地方！
F7 编译并运行

不知不觉放出了大招！！
